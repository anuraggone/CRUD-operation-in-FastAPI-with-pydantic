# CRUD Operations with FastAPI & Pydantic

This project is a **simple CRUD (Create, Read, Update, Delete) application** built using **FastAPI** and **Pydantic**.  
It was created **for learning purposes** to understand how basic CRUD operations work in FastAPI along with **data validation** using Pydantic models.

Data is stored in **patients.json** file
